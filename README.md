隱私權政策

最後更新日期：2025-12-18

歡迎使用【Easymaker123】（以下簡稱「本網站」）。
本網站為線上影片製作工具，並非常重視使用者的隱私權。

本網站使用 Google API 服務（包含 Google Picker API 與 Google Drive API），
僅在使用者明確授權並主動操作的情況下，存取其 Google Drive 資料。

────────────────────

一、蒐集的資料

1. Google 使用者資料  
當使用者透過 Google Picker 從 Google Drive 匯入素材時，
本網站僅存取使用者所選取的檔案及其必要的中繼資料
（例如檔名、檔案 ID、檔案類型）。

本網站不會存取使用者未主動選取的其他 Google Drive 檔案。

2. 技術性資訊  
為確保網站正常運作，系統可能會自動蒐集基本技術資訊，
包含 IP 位址、瀏覽器類型與裝置資訊。

────────────────────

二、資料使用方式

蒐集之資料僅用於以下目的：
- 提供影片製作與素材匯入功能
- 依使用者操作處理與顯示檔案
- 維護與改善本網站服務

資料不會用於廣告、行銷或任何與功能無關之用途。

────────────────────

三、Google 使用者資料政策說明（重要）

本網站對 Google 使用者資料之使用，完全遵守
Google API Services User Data Policy。

- 不會將 Google 使用者資料用於廣告用途
- 不會出售、出租或分享 Google 使用者資料
- Google Drive 資料僅於功能執行期間使用，
  不會永久儲存於本網站伺服器（或僅暫存於處理期間）

────────────────────

四、第三方服務

本網站僅使用 Google 提供之官方 API 服務，
不會將使用者資料提供給其他第三方。

────────────────────

五、資料安全

本網站採取合理的技術與管理措施，
以保護使用者資料免於未經授權的存取、洩漏或竄改。

────────────────────

六、使用者權利

使用者可隨時：
- 透過 Google 帳戶設定撤銷本網站的存取權限
- 停止使用本網站服務
- 聯絡我們要求刪除相關資料

────────────────────

七、隱私權政策更新

本政策如有更新，將公布於本頁面，
恕不另行個別通知。
